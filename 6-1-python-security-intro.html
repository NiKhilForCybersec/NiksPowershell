Why Python for Security?

Python has become the de facto language for security professionals due to its versatility, extensive library ecosystem, and cross-platform capabilities.

Setting Up Python Environment

Windows Installation

PowerShell - Install Python on Windows 
# Using winget (Windows 10+)
winget install Python.Python.3.12

# Verify installation
python --version
pip --version

# Add to PATH if needed (run as admin)
$env:PATH += ";$env:LOCALAPPDATA\Programs\Python\Python312;$env:LOCALAPPDATA\Programs\Python\Python312\Scripts"

Linux Installation

Bash - Install Python on Linux 
# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip python3-venv

# RHEL/CentOS/Rocky
sudo dnf install python3 python3-pip

# Verify installation
python3 --version
pip3 --version

# Create alias (add to ~/.bashrc)
echo "alias python='python3'" >> ~/.bashrc
echo "alias pip='pip3'" >> ~/.bashrc
source ~/.bashrc

Virtual Environments

Always use virtual environments to isolate project dependencies and avoid conflicts.

‚úÖ
Best Practice

Create a dedicated virtual environment for each security project. This prevents dependency conflicts and makes your tools reproducible.

Python - Virtual Environment Setup 
# Create virtual environment
python -m venv security-tools

# Activate (Windows)
.\security-tools\Scripts\Activate.ps1

# Activate (Linux/Mac)
source security-tools/bin/activate

# Verify activation (prompt shows venv name)
which python  # Should show venv path

# Deactivate when done
deactivate

Your First Security Script

Python - Basic Security Info Collector 
#!/usr/bin/env python3
"""
Basic Security Information Collector
Demonstrates Python security scripting fundamentals
"""

import socket
import platform
import getpass
import os
from datetime import datetime

def get_system_info():
    """Collect basic system information"""
    return {
        "hostname": socket.gethostname(),
        "platform": platform.system(),
        "platform_version": platform.version(),
        "architecture": platform.machine(),
        "processor": platform.processor(),
        "current_user": getpass.getuser(),
        "timestamp": datetime.now().isoformat()
    }

def get_network_info():
    """Collect basic network information"""
    hostname = socket.gethostname()
    try:
        local_ip = socket.gethostbyname(hostname)
    except socket.gaierror:
        local_ip = "Unable to determine"
    
    return {
        "hostname": hostname,
        "local_ip": local_ip,
        "fqdn": socket.getfqdn()
    }

def check_common_ports(target="127.0.0.1", ports=[22, 80, 443, 3389, 445]):
    """Check if common ports are open on target"""
    results = {}
    for port in ports:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(1)
        result = sock.connect_ex((target, port))
        results[port] = "Open" if result == 0 else "Closed"
        sock.close()
    return results

def main():
    print("=" * 50)
    print("Security Information Collector")
    print("=" * 50)
    
    # System Info
    print("\n[+] System Information:")
    sys_info = get_system_info()
    for key, value in sys_info.items():
        print(f"    {key}: {value}")
    
    # Network Info
    print("\n[+] Network Information:")
    net_info = get_network_info()
    for key, value in net_info.items():
        print(f"    {key}: {value}")
    
    # Port Scan
    print("\n[+] Common Ports (localhost):")
    ports = check_common_ports()
    for port, status in ports.items():
        print(f"    Port {port}: {status}")

if __name__ == "__main__":
    main()

Interview Tip

When asked about Python for security, mention: cross-platform compatibility, extensive library ecosystem (scapy, pefile, requests), rapid prototyping capability, and integration with tools like Splunk, Elastic, and cloud platforms.

Key Takeaways

  • Python excels at API integration, data analysis, and cross-platform tools
  • Always use virtual environments for project isolation
  • Key libraries: requests, scapy, paramiko, pefile, pandas
  • Use requirements.txt to manage dependencies
  • Python complements PowerShell/Bash rather than replacing them
