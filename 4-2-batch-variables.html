4.2 Batch Variables & Parameters

üìñ
What You'll Learn
  • Setting and using variables with SET
  • Environment variables and system paths
  • Command-line parameters (%1, %2, etc.)
  • Delayed expansion for loops
  • String manipulation techniques

Setting Variables

In Batch, variables are set using the SET command. Note: no spaces around the equals sign!

Batch - Basic Variables 
@echo off

REM Setting variables - NO SPACES around =
SET name=John
SET age=25
SET path_to_logs=C:\Windows\System32\winevt\Logs

REM Using variables with %VAR%
echo Hello, %name%!
echo You are %age% years old.

REM WRONG - spaces will be included in value!
REM SET name = John    <-- "name " = " John"

REM Variables are case-insensitive
echo %NAME% is same as %name%
‚ö†Ô∏è
Critical: No Spaces!

SET VAR=value is correct. SET VAR = value creates a variable named "VAR " with value " value" (including spaces). This is a common source of bugs.

User Input

Batch - Getting User Input 
@echo off

REM Prompt for input with /P
SET /P username=Enter your username: 
SET /P target_ip=Enter IP to scan: 

echo You entered: %username%
echo Target IP: %target_ip%

REM With default value pattern
SET /P choice=Continue? (Y/N) [Y]: 
IF "%choice%"=="" SET choice=Y

Numeric Variables

Batch - Arithmetic with /A 
@echo off

REM SET /A for arithmetic operations
SET /A count=0
SET /A count=%count%+1
SET /A count+=1

REM Math operations
SET /A result=10+5
SET /A result=20-8
SET /A result=6*7
SET /A result=100/4
SET /A remainder=17%%5    <-- Use %% for modulo in scripts

echo Count: %count%
echo Result: %result%
echo Remainder: %remainder%

REM Complex expressions
SET /A total=(10+5)*2
SET /A bits=1^<^<8    <-- Bitwise left shift (256)

Environment Variables

Batch - Common Environment Variables 
@echo off

REM System environment variables
echo Computer: %COMPUTERNAME%
echo Username: %USERNAME%
echo Domain: %USERDOMAIN%
echo User Profile: %USERPROFILE%
echo Temp Folder: %TEMP%
echo System Root: %SYSTEMROOT%
echo Program Files: %PROGRAMFILES%
echo Program Files x86: %PROGRAMFILES(x86)%

REM Path-related
echo System Drive: %SYSTEMDRIVE%
echo Home Drive: %HOMEDRIVE%
echo Home Path: %HOMEPATH%

REM Date and Time (format varies by locale!)
echo Date: %DATE%
echo Time: %TIME%

REM Processor info
echo Architecture: %PROCESSOR_ARCHITECTURE%
echo Processors: %NUMBER_OF_PROCESSORS%
üß†
Security Note

Malware often uses environment variables like %TEMP%, %APPDATA%, and %USERPROFILE% to write files. Understanding these helps with threat hunting and forensics.

Command-Line Parameters

Batch - Script Parameters 
@echo off
REM Save as: scanner.bat
REM Usage: scanner.bat 192.168.1.1 445

REM %0 = script name
REM %1 = first parameter
REM %2 = second parameter
REM %* = all parameters

echo Script: %0
echo Target IP: %1
echo Port: %2
echo All args: %*

REM Check if parameter provided
IF "%1"=="" (
    echo Usage: %0 ^<IP^> ^<PORT^>
    exit /b 1
)

REM Parameter modifiers
echo Full path: %~f0
echo Drive: %~d0
echo Path only: %~p0
echo Name only: %~n0
echo Extension: %~x0
echo Short name: %~s0
Modifier Description Example
%~1 Remove quotes "file.txt" ‚Üí file.txt
%~f1 Full path C:\Scripts\file.txt
%~d1 Drive letter C:
%~p1 Path only \Scripts\
%~n1 Filename file
%~x1 Extension .txt
%~dp1 Drive + Path C:\Scripts\
%~nx1 Name + Extension file.txt

Delayed Expansion

Variables inside loops need delayed expansion to get updated values:

Batch - Delayed Expansion 
@echo off
SETLOCAL EnableDelayedExpansion

REM Without delayed expansion - WRONG
SET count=0
FOR %%i IN (1 2 3 4 5) DO (
    SET /A count=%count%+1
    echo Count is: %count%    <-- Always shows 0!
)

REM With delayed expansion - CORRECT  
SET count=0
FOR %%i IN (1 2 3 4 5) DO (
    SET /A count=!count!+1
    echo Count is: !count!    <-- Shows 1,2,3,4,5
)

REM Use !VAR! instead of %VAR% inside loops
SET list=
FOR %%f IN (*.txt) DO (
    SET list=!list! %%f
)
echo Files: !list!

ENDLOCAL
‚úÖ
Rule of Thumb

Use %VAR% for reading variables outside loops. Use !VAR! with SETLOCAL EnableDelayedExpansion when reading/modifying variables inside loops or IF blocks.

String Manipulation

Batch - String Operations 
@echo off
SETLOCAL EnableDelayedExpansion

SET str=Hello World Security

REM Substring: %VAR:~start,length%
echo First 5: %str:~0,5%         <-- Hello
echo Skip 6: %str:~6%            <-- World Security
echo Last 8: %str:~-8%           <-- Security
echo Middle: %str:~6,5%          <-- World

REM String replacement: %VAR:old=new%
SET path=C:\Users\Admin\Documents
echo %path:\=/%                   <-- C:/Users/Admin/Documents
echo %path:Admin=User%            <-- C:\Users\User\Documents

REM Remove substring
SET filename=report.txt.bak
echo %filename:.bak=%             <-- report.txt

REM String length (tricky in batch)
SET str=TestString
SET len=0
:loop
IF NOT "!str:~%len%,1!"=="" (
    SET /A len+=1
    GOTO loop
)
echo Length: %len%

ENDLOCAL

Special Variables

Batch - Special Variables 
@echo off

REM Error level from last command
ping -n 1 192.168.1.1 >nul 2>&1
echo Exit code: %ERRORLEVEL%

REM Random number (0-32767)
echo Random: %RANDOM%
SET /A dice=%RANDOM% %% 6 + 1
echo Dice roll: %dice%

REM Current directory
echo Current dir: %CD%

REM Command extensions version
echo CMD version: %CMDEXTVERSION%

REM Dynamic date/time variables
echo Today: %DATE%
echo Now: %TIME%

REM For reliable date format, use WMIC
FOR /F "tokens=2 delims==" %%I IN ('wmic os get localdatetime /value') DO SET datetime=%%I
SET year=%datetime:~0,4%
SET month=%datetime:~4,2%
SET day=%datetime:~6,2%
echo Formatted: %year%-%month%-%day%

Security Script Example

Batch - System Info Collector 
@echo off
SETLOCAL EnableDelayedExpansion

REM Security info collector script
SET output=%TEMP%\sysinfo_%COMPUTERNAME%.txt

echo ========================================= > "%output%"
echo System Information Report >> "%output%"
echo Generated: %DATE% %TIME% >> "%output%"
echo ========================================= >> "%output%"

echo. >> "%output%"
echo [SYSTEM IDENTITY] >> "%output%"
echo Computer: %COMPUTERNAME% >> "%output%"
echo User: %USERDOMAIN%\%USERNAME% >> "%output%"
echo Architecture: %PROCESSOR_ARCHITECTURE% >> "%output%"

echo. >> "%output%"
echo [ENVIRONMENT] >> "%output%"
echo SystemRoot: %SYSTEMROOT% >> "%output%"
echo Temp: %TEMP% >> "%output%"
echo UserProfile: %USERPROFILE% >> "%output%"

echo. >> "%output%"
echo Report saved to: %output%
notepad "%output%"

ENDLOCAL

Key Takeaways

  • SET VAR=value - No spaces around the equals sign
  • %VAR% to read variables, !VAR! in loops with delayed expansion
  • SET /P for user input, SET /A for arithmetic
  • %1, %2 for command-line parameters
  • Use modifiers like %~dp0 to extract path components
  • Enable delayed expansion for loops: SETLOCAL EnableDelayedExpansion
