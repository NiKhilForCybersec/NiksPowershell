The Right Tool for the Right Job

Selecting the appropriate scripting language isn't just about personal preferenceâ€”it's about matching your tool to the task at hand. The wrong choice can lead to unnecessary complexity, poor performance, or maintenance headaches down the road.

Memory Aid: Tool Selection
  • PowerShell = "Swiss Army Knife" â†’ Object-rich, .NET powered, Windows-native
  • Bash = "Unix Toolbox" â†’ Text streams, pipes, Linux/Mac native

PowerShell: Active Directory User Audit

PowerShell's native AD module makes it the clear choice for Windows domain operations:

ğŸ”· PowerShell 
# Find inactive AD users (no login in 90 days)
$InactiveDate = (Get-Date).AddDays(-90)

Get-ADUser -Filter {LastLogonDate -lt $InactiveDate} `
    -Properties LastLogonDate, Department |
    Select-Object Name, SamAccountName, LastLogonDate, Department |
    Export-Csv -Path "InactiveUsers.csv" -NoTypeInformation

Bash: Log File Analysis

Bash excels at text processing with its powerful pipeline and standard tools:

ğŸ§ Bash 
#!/bin/bash
# Find top 10 IPs with failed SSH attempts

grep "Failed password" /var/log/auth.log | \
    grep -oE '[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+' | \
    sort | uniq -c | \
    sort -rn | \
    head -10 | \
    while read count ip; do
        echo "$ip: $count failed attempts"
    done

Python: API Integration with VirusTotal

Python's rich library ecosystem makes it ideal for API integrations:

ğŸ Python 
import requests
import hashlib

def check_file_hash(file_path, api_key):
    # Calculate SHA256 hash
    with open(file_path, 'rb') as f:
        file_hash = hashlib.sha256(f.read()).hexdigest()
    
    # Query VirusTotal API
    url = f"https://www.virustotal.com/api/v3/files/{file_hash}"
    headers = {"x-apikey": api_key}
    
    response = requests.get(url, headers=headers)
    if response.status_code == 200:
        data = response.json()
        stats = data['data']['attributes']['last_analysis_stats']
        return stats
    return None
